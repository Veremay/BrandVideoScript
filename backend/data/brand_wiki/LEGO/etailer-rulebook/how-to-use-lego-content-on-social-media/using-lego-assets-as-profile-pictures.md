## Source Content
The LEGO logo and other LEGO trademarks must never be used in or as part of your profile pictures on any social media channel. See ill. 61.

\*Only LEGO Certified Store partners would be allowed to use the LEGO brand logo (unaltered nor adapted from its original square format) as their profile picture, as long as the profile's cover image includes the LEGO logo and "Certified Store" annotation or partner's own logo if they have one. This is to differentiate that the profile is not run by the LEGO Group. All LEGO Certified Store related pages and profiles names should include "Certified Store" + "location/entity name" if they are to include LEGO word.
