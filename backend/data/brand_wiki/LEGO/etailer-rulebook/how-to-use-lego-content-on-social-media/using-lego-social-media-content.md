## Source Content
You can share content from the LEGO® Social Media channels on your own social media channels. Ensure that you are clicking 'Share' from the original post on the LEGO channel. Do not create a new post with the same content. If a post cannot be 'shared' for a given reason, we can provide you with the original assets. This applies to 'retweeting' on X platform and 'repining' on Pinterest. Always tag LEGO usernames where possible. See ill. 59-60.
